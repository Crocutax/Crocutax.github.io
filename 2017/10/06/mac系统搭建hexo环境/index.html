安装hexo

使用hexo -version指令测试hexo是否安装成功

初始化Hexo目录

创建自己的博客文件夹,然后cd到该目录下

1.执行指令hexo init初始化hexo

2.安装npm

输入指令:npm install

3.启动hexo本地服务

此时,打开http://localhost:4000/就可以看到hexo的默认首页了,hexo搭建完毕!

注意事项

hexo初始化的目录必须是空目录,所以如果是文件迁移,那么先hexo初始化空目录,然后再文件覆盖即可.
