明月几时有

Introduction

1941年末，香港沦陷。日军侵占香港疯狂抢掠，搜捕滞留在香港的中国文化人士，强令他们合作。香港东江游击队接到上级指示，紧急营救躲避在香港、九龙的文化人士，并在香港人民的掩护救助下，全部安全撤离香港。之后通过枪战、巷战、炸药、硝烟、鲜血和牺牲，讲述在香港沦陷的三年零八个月日子里，一群地道的香港小人物为保卫香港与凶残的日军抗争，奉献出青春和生命的真实故事

Note

  • 根据真人真事改编
  • 方兰从一个喜欢文学的小学老师转变为直面生死的市队游击队长,从一个对爱情充满向往的小女生转变成一个为了更多人的生命安危和革命事业而放弃了营救亲人的战士.就像电影里说的,环境可以改变人.
  • 无论是方兰,还是他身边的一男一女两个战友,诸多细节都展现出了他们还都只是小孩子.
  • 方兰,李锦荣,刘黑仔,在战争的时代背景下,对感情是向往的,但是都在理性的克制.都在期望,胜利后再见,虽然谁也不知道胜利何时到来,以及自己能否活到胜利之时.

游戏规则

Introduction

二战时期内忧外患的背景下，各方势力都想颠覆游戏规则挣脱宿命牢笼的故事.

Note

  • 了解规则–>熟悉规则–>企图颠覆规则 , 想必是野心家的必经路线,然而之所以称之为”游戏规则”,是因为每个人,包括企图颠覆者,其实都只是规则中的一员,这种行为本身就在规则之内,所以是注定失败的.
  • 唐先生还是太狠了,为了活下来不择手段不信任任何人,谁愿意跟他一起打江山?
  • 游戏里还是个人英雄主义严重,不太真实,李子豪(何润东饰演)太能打了,独挑青帮老巢,开玩笑呢.而且何润东依然以《风云-雄霸天下》中步惊云的形象和行为方式登场,改不了这个毛病了是吗?

破局

Introduction

这是高见翔警官生命中最艰难的一天：老母亲去世，疑似受贿被要求配合调查取证。在奔丧的路上，他又意外撞死了人。手忙脚乱的“毁尸灭迹”，他以为生活可以回归到波澜不惊。却不料，他撞死的其实是一名通缉犯，而驾车逃逸的他也同样被警方盯上。与此同时，一个神秘电话打来，高警官惊觉，这一切竟然是个阴谋，而他已经深陷迷局

Note

  • 母亲去世,灰色收入被查,交通肇事,开头的布局非常应景的交代了电影的主题《破局》,简直是无路可逃.
  • 不同于以往的高级警官形象,郭富城在这里面扮演的只是一个普通小警察,但是即便如此,也不应该被编剧写成一个见到陈建昌就立马爬着走,各种屁滚尿流鼻涕四溢,连一个被炸弹炸过的人都不敢pk这种软蛋形象.这好像连普通人都不如.
  • 头可断,郭天王发型永远不能乱.
  • 最后一幕是高见翔把大量的财富充公了,这一镜头是后加的吧?从剧情上来讲根本不通,可能是为了能顺利通过有关部门的审批吧.

辑枪

Introduction

一次普通的车辆剐蹭事故里，俞苗老公刘明义拿枪打死了徐天（白举纲饰）的女朋友，被警察当场击毙。徐天找俞苗（热依扎饰）赔钱，二人去找卖枪给刘明义（任东霖饰）的枪贩子马三叹（洪剑涛饰），却遇上了马三叹被警察连广阔抄了老窝。马三叹跳桥重伤，俞苗和徐天被马三叹的弟弟小马（王泷正饰）绑架，小马用徐天的生命要挟俞苗，要她去医院杀马三叹灭口。俞苗到医院，在连广阔的帮助下蒙混过关，警方根据俞苗的线索追查小马下落，警方设卡在山路围捕抢匪，双方展开了最后的较量。

Note

  • 多亏了美女热依扎称场面,否则整场电影会更加的枯燥乏味.
  • 外卖小哥徐天为什么能买公墓?有钱吗?死者的假身份证也可以审核通过吗?为什么他在知道女友只把女友当提款机的情况下,还要全程追着要15万?
  • 为什么警察连广阔在中枪后,一直盯着俞苗看,没明白.
  • 一切事情,只要讲规矩,都好办.
  • 中美国情差距大,在中国开放枪支还是相当危险的,社会治安和政局都会陷入巨大的混乱.
